Data mutation means changing existing data directly after it has been created.
It can cause unexpected behavior if other parts of the program rely on the original data.

**GET, POST, PUT, DELETE** are HTTP methods used in web development:

* **GET**: Retrieves data from a server.
* **POST**: Sends new data to the server to create something.
* **PUT**: Updates existing data on the server.
* **DELETE**: Removes data from the server.
